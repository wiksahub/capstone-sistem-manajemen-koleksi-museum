# Capstone Modul 1: Sistem Manajemen Koleksi Museum
Aplikasi sederhana sebagai capstone modul 1. Sistem Manajemen Koleksi Museum menggunakan python.
Aplikasi ini mengimplementasi fitur CRUD menggunakan python.
## 1. Read
Pada menu read, pengguna dapat menjalankan beberapa opsi. 

a. Melihat keseluruhan koleksi museum
Pada opsi ini, pengguna dapat melihat koleksi museum secara menyeluruh serta jumlah item yang ada dalam koleksi serta nilai estimasi dari keseluruhan koleksi.

b. Melakukan pencarian berdasarkan kategori
Pada opsi ini, pencarian dilakukan berdasarkan kategori dimana hanya ditampilkan kategori yang diinput.

c. Melakukan pencarian berdasarkan ID item
Pada opsi ini, aplikasi akan menampilkan satu item dengan ID yang telah diinput.

## 2. Create
Pada menu ini, pengguna dapat menambahkan entry / item baru dimana ID pada item baru akan dibuat dengan format: tahun + kategori + ID unik 3 digit

## 3. Delete
Pada menu ini, pengguna ditampilkan keseluruhan koleksi museum dan diminta untuk input ID item yang akan dihapus.

## 4. Update
Pada menu ini, pengguna dapat melakukan perubahan pada kategori atau kondisinya. Jika yang diubah kategori dari item, ID dari item tersebut akan berubah sesuai kategori yang diganti.
